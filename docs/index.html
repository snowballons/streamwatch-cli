How to Use

Getting started with StreamWatch is simple:

  1. Install StreamWatch: 
    pip install streamwatch
  2. Prerequisites: Ensure you have Python (3.7+), Streamlink, and MPV Media Player installed and in your PATH.
  3. Run StreamWatch: 
    streamwatch
  4. Add Streams: On first run, or using the 'A' option, add your favorite stream URLs.
  5. Navigate & Play: Use the interactive menu (arrow keys or number input) to select and play live streams.

For detailed commands and options, check out the full documentation.
